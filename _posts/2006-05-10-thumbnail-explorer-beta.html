We're happy to announce the beta release of a new program called Thumbnail Explorer. The software is similar to our original Magic File program but this application is focused at viewing and searching for visual files. The software is a visual search engine for your PC.

  • Quickly search for thumbnails, graphic files and CAD files quickly
  • Index thousands of folders and search them instantly.

Users can download a free trial version of Thumbnail Explorer.
