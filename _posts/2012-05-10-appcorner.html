In this week's episode of AppCorner we talk about the Google vs. Oracle lawsuit, discuss Evernote's acquiring of iPad app Penultimate, chat Facebook's upcoming IPO, and our app recommendations (and fails) of the week!

AppCorner is hosted by the team at Raizlabs. This week's podcast crew included Greg Raiz (@graiz), Dave Counts (@davecounts), Craig Spitzkoff (@oofoot), Bruno Germansderfer (@brunog) and Michelle Buchman (@tres_michelle).

[display_podcast]

Apps discussed in this episode include:

  • Due - creates reminders based on events from Agenda Calendar.
  • Clear - a to-do list app with beautiful UI for iPhone.
  • NPR Music - Engage, explore and discover new music of all genres with the NPR Music app.
  • LiveView - a specialized remote screen viewing application intended as a tool to help designers create graphics for mobile applications.
  • PaintCode- a simple vector drawing app that instantly generates resolution-independent Objective-C drawing code for both OS X and iOS.
  • Adobe Shadow - a new inspection and preview tool that allows front-end web developers and designers to work faster and more efficiently by streamlining the preview process.
  • WebEx - users can join any web conference right from their iPad or iPhone.
