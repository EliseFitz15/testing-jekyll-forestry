A new version of Thumbnail Explorer has been released with several improvements including support for x64 as well as a translation into French. The Thumbnail Explorer software is designed to search for images, graphic files and CAD application files. The software acts as a desktop search engine making it easy to quickly find graphical files. A trial version of the software is available.
